HTML이란?

HTML은 HyperText Markup Language의 약자로 웹페이지를 구성하는 마크업 언어입니다.
자바 스크립트와 CSS 등을 같이 사용하여 본문과 외관의 배치를 정의할 수 있습니다.

HTML의 기초문법

HTML은 기본적으로 요소로 구성 되어 있습니다. 요소는 시작 태그와 종료 태그의 쌍으로 구성 되어 있으며, 태그는 <>(꺾쇠 괄호)로 둘러싸인 키워드입니다.


HTML의 기초문법 예시
    <head>

        <title>제목</title>
    
    </head>

HTML의 주석

다른 언어들과 같이 HTML에서도 주석을 사용할 수 있습니다. 주석은 코드를 이해하는데 도움을 주며, 웹 페이지에는 표시되지 않습니다.

주석은 <!-- 로 시작해서 --> 로 끝나며, 사이에 설명을 적어놓으면 됩니다. 아래는 주석 사용 예시입니다.


    <!-- 이것은 주석입니다. -->
