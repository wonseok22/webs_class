CSS란?

CSS는 Cascading Style Sheets의 약자로 HTML, XHML, XML 같은 문서의 스타일을 꾸밀 때 사용하는 스타일 시트 언어이다.


HTML로 문서의 뼈대를 만들면 CSS는 이 문서를 꾸며 글꼴, 배경색, 위치, 너비와 높이 등을 지정하거나 웹 브라우저, 스크린 크기, 장치에 따라서 화면을 다르게 표시될 수 있도록 지정할 수 있다.

CSS 주석 표시

CSS의 주석은 /*과 */기호를 사용하며 기호 사이에 주석 내용을 적으면 된다.

/* 이곳에 내용을 적으세요 */

CSS 선언 방법

내부 스타일

HTML 문서 내에서 <head>와 </head>사이에 <style> 태그로 지정하는 방법

<head>
    <style type="text/css">
        Selector {property: value;}
       ex) h1 {color:red} 
    </style>
</head>

외부 스타일

CSS라는 확장자를 가진 스타일 시트 파일을 만들고 이 파일을 HTML 문서에 연결하여 사용하는 방법

<head>
        <link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

인라인 스타일

스타일을 지정하고싶은 태그에 직접적으로 주입해주는 방식, style 속성으로 property: value 의 값을 가짐

<p style="font-size: 15px;">15픽셀사이즈로 디자인됩니다.</p>
