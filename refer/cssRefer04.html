CSS 색상이란?

CSS 색상을 표현하는 방법

1. 색상 이름으로 표현

색상 이름으로 표현이 가능하다. W3C에서 정의한 16개의 HTML4 표준 색상 이름은 다음과 같다.

aqua black blue fuchsia
gray green lime maroon
navy olive purple red
silver teal white yellow
색상 이름 사용 예시이다.
<style>
    .blue { color : blue; }
    .green { color : green; }
    .silver { color : silver; }
</style>

2. RGB 색상값으로 표현

R (RED), G(GREEN), B(BLUE) 세 가지 색의 혼합을 표현한다. 0부터 255 사이의 범위를 가지며 이를 (R,G,B)로 표현한다.

RGB표현 사용 예시이다.
<style>
    .blue { color: rgb(0,0,255); }
    .green { color: rgb(0,128,0); }
    .silver { color: rgb(192,192,192); }
</style>

3. 16진수 색상값으로 표현

16진수 색상값은 RGB 색상값을 각각 16진수로 변환한 것이다. 따라서 RGB 색상의 기본색(Red, Green, Blue)은 각각 00부터 FF까지의 범위를 가진다.

16진수표현 사용 예시이다.
<style>
    .blue { color: #0000FF; }
    .green { color: #008000; }
    .silver { color: #C0C0C0; }
</style>
