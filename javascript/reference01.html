자바스크립트 선언문

선언문 : 자바스크립트 코드를 작성할 영역을 선언하는 것

선언문은 <head>태그 영역 또는 <body> 태그 영역에 선언하면 된다.

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Document＜/title>
    <script>
        document.write("환영합니다")
    </script>
</head>
<body>
</body>
<html>

자바스크립트 주석처리

자바스크립트 선언문 안에서 코드가 아닌 설명글을 넣고 싶을 때는 주석 처리를 하면 됩니다.

이때 한 줄 주석일 경우에는 '//한줄 설명글'로 작성하고, 여러 줄 주석 일 경우에는 '/*여러줄 설명글*/'로 작성하면 됩니다.

html 주석방법은  <!-- --> 입니다.  ex)<!-- HTML 주석처리 -->   단축키는 Ctrl + / 입니다. 
자바스크립트 주석 방법은  한줄일 경우 // 입니다 ex) //Javascript 주석처리
두줄 일경우 /* */을 사용해 줍니다. ex) /*  Javascript
                                                         두줄 주석처리 */

내부 스크립트 외부로 분리하기

HTML 내부에 작성된 자바스크립트는 마지막에는 외부로 분리하는 작업을 해야 합니다. 그렇게 하면 자바스크립트 소스 찾기도 쉬울 뿐만 아니라 누군가가 소스를 손상시킬 염려도 없습니다. 즉, 외부로 분리하는 주된 목적은 프로젝트 관리를 원활하게 하기 위함입니다. 다음은 외부 자바스크립트 연동 코드로 파일을 연결하는 기본형입니다.

{
    <script src="js 파일 경로"></script>
}

{
    <!DOCTYPE html>
    <html lang="ko">
        <head>
            <meta charset="UTF-8">
            <title>외부 자바스크립트 연동</title>
            <scrips src="자바스크립트파일.JS"></scrips>
        </head>
        <body>
        </body>
    </html>
}

코드 입력시 주의 사항

자바스크립트는 대·소문자를 구분하여 작성합니다.

ex) 날짜 객체 생성 : New date(); (X)
ex) 날짜 객체 생성 : new Date(); (O)

1. 코드를 한 줄을 작성한 후에는 세미콜론을 쓰는 것이 좋습니다.

2. 세미콜론을 쓰지 않고 한 줄에 2개의 코드를 작성할 경우 오류가 발생합니다.

ex) document.write("hi") document.write("bye")  (X)
ex) document.write("hi"); document.write("bye");  (O)

3. 코드를 작성 할때는 한줄에 한 문장만 작성하는 것이 가독성을 위해 좋습니다.

4. 문자형 데이터를 작성할 때는 큰따옴표("")와 작은따옴표('')의 겹침 오류를 주의해야 합니다.

큰따옴 겹칩 오류
document.write("책에 "자바스크립트는 대소문자를 구분해야 합니다"라고 나와 있다.");
올바르게 사용된 예제
document.write('책에 "자바스크립트는 대소문자를 구분해야 합니다"라고 나와 있다.');
document.write("책에 \"자바스크립트는 대소문자를 구분해야 합니다\"라고 나와 있다.");

5. 코드를 작성할 때 중괄호{} 또는 소활호()의 짝이 맞아야 합니다.

document.write("welcome!"; (X) 
document.write("welcome!"); (O)
