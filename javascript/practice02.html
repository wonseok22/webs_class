1. 구구단 출력하기

1.1 for 문을 사용해서 1단부터 9단까지 출력하시오

{
    //for문
    for(let i=1; i<10; i++){
        for(let j=1; j<10; j++){
            console.log(i + " * " + j + " = " , i*j);
        }
    }
}

1.2 while문을 사용해서 1단부터 9단까지 출력하시오

{
    let m = 1;
    while (m<10){
        let n = 1;
        while (n<10){
            console.log(m + " * " + n + " = " , m*n);
            n ++;
        }
        console.log('\n');
        m ++;
    }
}

조건 : 사용자가 원하는 단은 prompt()를 사용합니다. 클릭

{
    let num, result;
    
    document.querySelector(".btn03").addEventListener("click", () => {

        for(let i=1; i<10; i++){
            console.log(num + " * " + i + " = " + (num*i) + "\n";
        }
    })
}
